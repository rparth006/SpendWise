# 💰 SpendWise - Expense Tracker

SpendWise is a simple Python-based expense tracking application that helps users manage daily expenses, categorize spending, and monitor total budget usage.

## 🚀 Features
- Add daily expenses
- Category-wise tracking (Food, Travel, etc.)
- View all expenses
- Calculate total spending
- Data stored in JSON file

## 🛠 Tech Stack
- Python
- JSON file handling

## ▶️ How to Run
python expense_tracker.py

## 🎯 Purpose
Built for practicing Python file handling and real-world utility application development.
